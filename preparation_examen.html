Configuration Firebase requise

Pour synchroniser la progression sur tous les appareils, tu dois connecter une base de données Firebase gratuite. C'est une seule fois, prend ~5 minutes.

1
Va sur console.firebase.google.com et crée un projet (gratuit, nom libre)
2
Dans le projet → Build → Realtime Database → Create database → choisis "Start in test mode"
3
⚙️ Project settings → Your apps → </> → Enregistre l'appli → copie le bloc firebaseConfig
4
Ouvre preparation_examen.html, cherche FIREBASE_CONFIG et colle ta config dans les champs vides
5
Sauvegarde et pousse sur GitHub → le site est prêt pour tout le monde !
⚠️ Le mode test expire après 30 jours. Dans Firebase → Realtime Database → Rules, mets ".read": true, ".write": true pour rendre permanent.
